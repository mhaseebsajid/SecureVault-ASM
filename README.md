# 🔐 SecureVault-ASM

An Assembly Language-based Password Manager developed using MASM and the Irvine32 library as part of a Computer Organization and Assembly Language (COAL) semester project.

## 📌 Overview

SecureVault is a console-based password manager that securely stores user credentials using simple encryption techniques. The application requires a master PIN for authentication and demonstrates file handling, encryption, and modular programming in x86 Assembly Language.

## ✨ Features

- Master PIN Authentication
- Add New Credentials
- View Stored Credentials
- Update Existing Passwords
- Delete Credentials
- XOR & Caesar Cipher Encryption
- File-Based Storage
- Console-Based User Interface

## 🛠 Technologies Used

- MASM (Microsoft Macro Assembler)
- Irvine32 Library
- x86 Assembly Language
- Windows Console
